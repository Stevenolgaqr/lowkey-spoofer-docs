# Flashing Bios (VALORANT)

{% hint style="danger" %}
**FOR VALORANT ONLY!**
{% endhint %}

### Requirements:

* UPS or stable power supply
* USB 4GB+
* BIOS file for your motherboard
* Your phone to read instructions

### Steps:

1. Find your motherboard model
2. Download BIOS update from manufacturer website
3. Put BIOS file on USB (FAT32 format)
4. Enter BIOS
5. Use flash utility (EZ Flash, Q-Flash, M-Flash, etc.)
6. Select BIOS file from USB
7. Wait 3-5 minutes
8. PC will restart automatically

{% hint style="warning" %}
**DO NOT TURN OFF PC DURING FLASH!**
{% endhint %}
