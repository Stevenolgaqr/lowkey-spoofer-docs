# 4️⃣Download you Ethernet Drivers

{% hint style="info" %}
**ONLY FOR ETHERNET USERS!**

If you use WiFi, you'll need to purchase Ethernet adapter first.
{% endhint %}

***

## Why Download Drivers?

After reinstalling Windows WITHOUT internet connection, you'll need Ethernet drivers to connect online.

***

## Download Before Reinstalling Windows:

### Intel Ethernet Drivers

[Download Intel Ethernet Drivers](https://www.intel.com/content/www/us/en/download/15084/intel-network-adapter-driver-for-windows-10.html)

***

### Realtek Ethernet Drivers

[Download Realtek Ethernet Drivers](https://www.realtek.com/en/component/zoo/category/network-interface-controllers-10-100-1000m-gigabit-ethernet-pci-express-software)

***

### Killer Ethernet Drivers

[Download Killer Ethernet Drivers](https://www.killernetworking.com/driver-downloads/category/killer-suite)

***

### Aquantia Ethernet Drivers

[Download Aquantia Ethernet Drivers](https://www.marvell.com/support/downloads.html)

***

## Installation:

1. Download drivers to USB BEFORE reinstalling Windows
2. After Windows installation
3. Copy drivers from USB to PC
4. Install drivers
5. Restart PC
6. Ethernet should now work

***

{% hint style="success" %}
**Tip:** Download ALL common drivers just in case!
{% endhint %}
