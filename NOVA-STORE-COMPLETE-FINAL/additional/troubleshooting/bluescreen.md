Instructions on how to fix bluescreens during/after spoofing.

***

## Common Causes:

1. **Secure Boot enabled** in BIOS
2. **TPM conflicts**
3. **Corrupted Windows installation**
4. **Hardware issues**

***

## Solutions:

### Solution 1: Check BIOS Settings

1. Enter BIOS
2. **Disable Secure Boot**
3. **Disable TPM** (except for Valorant - keep enabled)
4. Save and exit

***

### Solution 2: Reinstall Windows

If bluescreens continue:
* Do a **clean Windows reinstall**
* Follow [Windows Reinstall Guide](../../spoofer-setup/windows-reinstall/)
* Delete all partitions during install

***

### Solution 3: Check Hardware

* Test RAM with **MemTest86**
* Check disk health with **CrystalDiskInfo**
* Update motherboard BIOS to latest version

***

{% hint style="warning" %}
**Persistent bluescreens?** May indicate hardware failure. Contact support.
{% endhint %}
