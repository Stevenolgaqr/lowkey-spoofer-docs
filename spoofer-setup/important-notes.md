**Why:** Old hardware traces remain in registry

**Why:** Secure Boot and TPM interfere with spoofer

**Fix:** Disable completely using our guide

### 5. Installing Windows Updates First
**Symptom:** Defender re-enables, drivers reinstall  
**Why:** Updates override your settings  
**Fix:** Disable updates, spoof first, then update if needed
