# 2️⃣BIOS Configurations

### How to Enter BIOS:

Press **Del** or **F2** during PC startup (when you see manufacturer logo)

***

## Step 1: DISABLE TPM (Trusted Platform Module)

{% hint style="danger" %}
**FOR VALORANT: KEEP TPM ENABLED!**
{% endhint %}

**For all other games, TPM must be DISABLED:**

> **ASUS:** [**https://streamable.com/sicp16**](https://streamable.com/sicp16)
>
> **INTEL 00:00 | AMD STARTS 01:20**

> **MSI:** [**https://streamable.com/n7q3dk**](https://streamable.com/n7q3dk)
>
> **INTEL 00:00 | AMD STARTS 01:05**

> **Asrock:** [**https://streamable.com/ec8u3s**](https://streamable.com/ec8u3s)
>
> **INTEL 00:00 | AMD STARTS 01:35**

> **GIGABYTE:** [**https://streamable.com/1qhn35**](https://streamable.com/1qhn35)
>
> **INTEL 00:00 | AMD STARTS 01:50**

***

## Step 2: DISABLE WiFi & Bluetooth in BIOS

{% hint style="warning" %}
**ONLY IF YOU HAVE WIFI/BLUETOOTH AVAILABLE!**

If your PC has NO WiFi/Bluetooth hardware, skip this step.
{% endhint %}

{% hint style="info" %}
If you ONLY have WiFi (no ethernet), you MUST purchase an Ethernet adapter or USB Ethernet dongle before using spoofer!
{% endhint %}

### By Motherboard Brand:

**ASUS:**
* Advanced > Onboard Devices Configuration
* Set WiFi (WLAN) to **Disabled**
* Set Bluetooth to **Disabled**

**MSI:**
* Advanced > Integrated Peripherals
* Set WiFi (WLAN) to **Disabled**
* Set Bluetooth to **Disabled**

**GIGABYTE:**
* Advanced / Peripherals
* Set WiFi (WLAN) to **Disabled**
* Set Bluetooth to **Disabled**

**ASROCK:**
* Advanced > Chipset Configuration
* Set WiFi (WLAN) to **Disabled**
* Set Bluetooth to **Disabled**

***

## Step 3: DISABLE Secure Boot

* Find "Secure Boot" in BIOS settings
* Set to **Disabled**

***

## Step 4: Save and Exit

* Press **F10** to save
* Confirm changes
* PC will restart

***

{% hint style="success" %}
**BIOS Configuration Complete!**
{% endhint %}
