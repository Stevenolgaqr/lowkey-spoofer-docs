## Step-by-Step Instructions

### 1. Extract Loader

* Extract the **Loader.rar** file
* Open the extracted folder

### 2. Run Setup Scripts

* Run **"RUN ME FIRST.bat"** as Admin
* Run **"RUN ME ALSO.bat"** as Admin

### 3. Open Loader

* Open **"Brave.exe"** or **"Loader.exe"** as Administrator

### 4. Login

* Enter your **License Key**
* Click **Activate**

### 5. Save Current Serials

* Go to **"Device Info"** tab
* Take screenshot of all serials

### 6. Permanent Spoof

* Go to **"Spoof"** tab
