# 6️⃣Spoof Monitor (Apex Legends)

{% hint style="warning" %}
**FOR APEX LEGENDS ONLY!**

Other games don't need this step.
{% endhint %}

***

## Why Spoof Monitor?

Apex Legends tracks your **monitor serial number**. This must be changed!

***

## Step 1: Disable G-SYNC

**If your monitor supports G-SYNC:**

![Disable G-SYNC](../../images/apex-monitor/disable-gsync.png)

1. Open **NVIDIA Control Panel**
2. Go to **"Set up G-SYNC"**
3. **Uncheck** "Enable G-SYNC"
4. Click **"Apply"**

***

## Step 2: Open Monitor Spoof Tool

1. Go to Nova Loader folder
2. Open **"Monitor Spoof"** folder
3. Run **CRU.exe** as Administrator

***

## Step 3: Follow Video Tutorial

{% hint style="info" %}
**WATCH CAREFULLY:** [**https://streamable.com/6ibjqa**](https://streamable.com/6ibjqa)
{% endhint %}

***

## Step 4: Change Serial Numbers

{% hint style="danger" %}
**IMPORTANT RULES:**
{% endhint %}

* ✅ Change **only 2-3 numbers**
* ✅ Replace **number for number** (5 → 8)
* ✅ Replace **letter for letter** (K → M)
* ❌ DON'T change entire serial
* ❌ DON'T use random characters

**Example:**
```
Before: XKA25679
After:  VGA62679
```

{% hint style="warning" %}
**If serial field is EMPTY - leave it empty!**
{% endhint %}

***

## Step 5: Apply Changes

1. Save changes in CRU
2. Run **restart64.exe** (in same folder)
3. **Screen will go black** - this is normal!
4. Wait 5 seconds
5. Screen returns ✅

***

## If Screen Stays Black:

Don't panic!

1. Unplug **HDMI/DisplayPort cable**
2. **Restart PC**
3. Replug cable after Windows boots
4. Should work now
